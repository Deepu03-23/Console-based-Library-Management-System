# Console-Based Library Management System

## Description
A Java-based console application to manage books, members, and borrowing operations using Object-Oriented Programming and file handling.

## Features
- Add, remove, search, borrow, and return books
- Member registration
- Due date tracking and overdue fine calculation
- Library statistics
- Export books to CSV
- File-based data persistence

## Technologies Used
- Java
- OOP Principles
- File I/O
- Collections (ArrayList)
- Java Time API

## How to Run
1. Open project in VS Code
2. Open `Main.java`
3. Click ▶ Run
4. Use menu options in terminal
